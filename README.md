# RExVicon
A simple interface to the Vicon SDK to relay data over ZMQ or serial


## Dependencies
Install the Boost library:

### Linux
```shell
sudo apt-get install libboost-all-dev
```

### Install on RPi3
Download ViconSDK
clone libserialport
```
sudo apt-get install autoconf
git clone git://sigrok.org/libserialport
cd libserialport
./autogen
./configu
```
